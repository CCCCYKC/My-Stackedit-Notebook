## 1. 全局方法的访问
Vue.js 在其原型链上定义了一些全局方法，可以通过实例对象或者全局引用来调用。例如，`$emit`、`$on` 等方法。


## 2. 实例属性的访问
在 Vue.js 2 中，实例对象上有一些特定的属性，可以通过原型链访问
```js
// 创建 Vue 实例
const vm = new Vue({
  data: {
    message: 'Hello, Vue!'
  }
});
 
// 访问实例的属性
console.log(vm.$data.message); // 输出: Hello, Vue!
```
