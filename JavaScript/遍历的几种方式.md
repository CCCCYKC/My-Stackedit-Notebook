# 遍历的几种方式
## for(let a in b)
`for...in`循环用于遍历**对象**的可枚举属性（包括继承的可枚举属性）。它会枚举出对象的键（即属性名）
```js
const obj = { a: 1, b: 2, c: 3 };
for (const key in obj) {
  console.log(key, obj[key]); // 输出键和值
}
```
## for(let a of b)
`for...of`循环用于遍历可迭代对象（如数组、字符串、`Map`、`Set`等），它会直接枚举出对象的值
